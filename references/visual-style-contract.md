# MDH Visual Style Contract

Este projeto deve preservar permanentemente esta assinatura visual como base da storefront:

- vídeo rodando no fundo das áreas hero e vitrines principais
- estética neon/cyber com ciano `#03e9f4` e violeta `#7b2cbf`
- superfícies escuras em glassmorphism
- hover com iluminação RGB perceptível em cards, painéis, tabs e botões
- clique com resposta visual forte, mantendo sensação de profundidade e pressão
- scan lines, glow e microanimações consistentes com a linguagem do hero

Regras de preservação:

- não migrar a base da UI para tema claro, bege ou institucional neutro
- não remover vídeo de fundo do hero como linguagem principal
- não reduzir os estados de hover/click para interações planas ou sem brilho
- novas seções podem variar layout, mas devem respeitar esta assinatura visual

Objetivo:

- manter a MDH 3D com leitura premium, energética e memorável, sem sacrificar clareza comercial nem usabilidade mobile
