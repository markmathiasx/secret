# MDH 3D Store — Kit de Execução Sem Codex

Este pacote foi montado para você aplicar as melhorias **manual e progressivamente**, mesmo sem Codex.

## O que este kit resolve
- Prioriza o que mexer primeiro
- Dá copy pronta para Home e confiança comercial
- Organiza curadoria do catálogo por intenção de compra
- Padroniza o template de produto
- Melhora FAQ / Entregas / Trocas
- Entrega referência de SEO com Product schema
- Entrega referência de imagens com `next/image`
- Entrega checklist de Vercel / domínio / SSL / redeploy
- Entrega referência de Docker + Node 24 para produção
- Entrega checklist final de aceitação antes de republicar

## O que este kit NÃO promete
- Não substitui edição real do código-fonte
- Não garante “100% concluído” sem abrir e testar o seu repositório completo
- Não valida páginas privadas se elas não forem fornecidas

## Como usar sem Codex
- Abra o arquivo correspondente ao bloco que quer corrigir
- Copie o trecho de referência
- Edite manualmente no seu VS Code
- Rode localmente
- Valide com o checklist final
- Só depois publique
