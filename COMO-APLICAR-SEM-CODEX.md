# Como aplicar sem Codex

## Método simples
1. Abra o projeto no VS Code
2. Escolha um item do backlog P0 ou P1
3. Copie o trecho de referência deste kit
4. Edite manualmente o componente / página correspondente
5. Rode localmente
6. Valide no navegador desktop e mobile
7. Só faça commit depois do checklist passar

## Ritmo sugerido
- Dia 1: deploy / Vercel / domínio
- Dia 2: Home + catálogo
- Dia 3: páginas de produto
- Dia 4: checkout + institucional
- Dia 5: SEO + imagens + revisão

## Regra de ouro
Não publique por “achar que ficou melhor”.
Publique somente quando o checklist final estiver 100% marcado e revisado.
