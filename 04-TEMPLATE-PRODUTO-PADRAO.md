# Template padrão de produto

## Campos obrigatórios
- Nome forte
- Slug
- Categoria
- Coleção
- Uso principal
- Material
- Dimensões
- Peso
- Personalização
- Prazo de produção
- Faixa de entrega
- Preço
- 3 benefícios
- 1 objeção respondida
- Foto principal
- Galeria
- Alt text

## Estrutura recomendada
### Nome
Curto, claro e comercial.

### Subheadline
Explique para quem serve e o diferencial.

### Benefícios
1. Visual / acabamento
2. Uso / utilidade
3. Personalização / presente

### Objeção respondida
Exemplo: “A peça é frágil?” → Resposta objetiva.

## Exemplo preenchido
- Nome: Luminária LED Personalizada de Mesa
- Categoria: Luminárias
- Uso principal: Presente criativo / decoração
- Material: PLA premium
- Dimensões: 18 x 12 x 8 cm
- Personalização: Nome / cor / frase
- Prazo de produção: 3 a 5 dias úteis
- Preço: R$ 89,90
- Benefícios:
  - visual exclusivo
  - ideal para presente
  - leve e fácil de posicionar
- Objeção respondida:
  - “A iluminação é fraca?” → É pensada para decoração ambiente, não leitura técnica.
